Vector Database with TensorFlow.js Demo

How It Works

This demo shows how vector databases work using TensorFlow.js and the Universal Sentence Encoder:

  1. Text is converted into high-dimensional vectors using the Universal Sentence Encoder
  2. These vectors capture the semantic meaning of the text
  3. Similar texts will have similar vector representations
  4. Vector similarity is calculated using cosine similarity
